# Streamlit Cloud Deployment

This app is configured for easy deployment on Streamlit Cloud.

## Quick Deploy Steps:

1. **Go to [share.streamlit.io](https://share.streamlit.io)**
2. **Connect your GitHub account**
3. **Select repository:** `armAAn12139/Portfolio-Intelligence`
4. **Set main file:** `app.py`
5. **Deploy!** 🚀

## Configuration:
- **Python version:** 3.8+
- **Main file:** `app.py`
- **Requirements:** `requirements.txt` (already configured)

## Features that work online:
- ✅ Interactive portfolio analysis
- ✅ Real-time market data fetching
- ✅ Plotly charts and visualizations
- ✅ What-if scenario planning
- ✅ AI insights and recommendations
